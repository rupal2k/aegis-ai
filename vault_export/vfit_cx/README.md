Standalone Obsidian vault for Vantage Fit client success operations.

- Reporting, billing, and active-user verification
- Internal collaboration with product, backend, data, marketing, and design

## Working rules

- Do not invent client metrics, renewal dates, or account details.
- If a metric is not confirmed, write `TBD` or request data.
- When client data is shared externally, prefer aggregated summaries over raw sensitive detail.
- Use the proof-point note before citing stats in client communication.
